# Decode JWTs

[![npm](https://img.shields.io/npm/v/jwt-d)](https://www.npmjs.com/package/jwt-d)

A small utility to decode JSON Web Tokens.

## Aliasing

Consider adding `alias jwt-d='npx jwt-d@latest'` to your shell.

That would allow you to directly invoke the latest version of `jwt-d` via:

```bash
jwt-d eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJzdWIiOiIxMjM0NTY3ODkwIiwibmFtZSI6IkpvaG4gRG9lIiwiaWF0IjoxNTE2MjM5MDIyfQ.
```
